# BabyGEO Pro 图片规范指南

## 一、整体摄影风格要求

### 1.1 通用拍摄标准
- **背景**: 纯白或浅灰色背景（#FFFFFF 或 #F5F5F5），禁止复杂背景
- **光线**: 柔和的自然光或专业影棚灯，避免硬阴影
- **色调**: 温暖柔和的中性色调，呈现专业B2B商业感
- **构图**: 产品居中，占画面70-80%，留白充足
- **锐度**: 高锐度，确保细节清晰可见
- **色彩还原**: 真实还原产品颜色，不过度饱和

### 2.1 图片格式
- **格式**: WebP优先，备选JPEG
- **质量**: 80-85%压缩率
- **命名**: `{类别}-{用途}.jpg`

## 五、后期处理规范

### 5.1 调色参数参考
```
亮度 (L*): 70-82
对比度: +5 to +15
饱和度: 0 to +10 (轻微增强)
色温: 5500K-6500K (中性日光)
```

### 5.2 锐化参数
```
数量: 150-180%
半径: 0.8-1.0px
阈值: 2-3 levels
```

### 5.3 压缩设置
```
JPEG: 品质80-85
分辨率: 72dpi (Web)
格式: WebP with JPEG fallback
```

## 七、禁止事项

1. ❌ 禁止使用卡通或插画风格的图片
2. ❌ 禁止使用水印或第三方Logo
3. ❌ 禁止使用拉伸变形或低分辨率图片
4. ❌ 禁止使用含有儿童模特的图片（涉及肖像权）
5. ❌ 禁止使用与产品无关的装饰元素
6. ❌ 禁止使用过曝或欠曝的图片
7. ❌ 禁止使用饱和度过高或过低（偏色）的图片
