## 🔧 配置文件说明

| 文件 | 作用 | 说明 |
|------|------|------|
| `.env` | 环境变量 | 数据库连接、API 密钥等敏感信息 |
| `docker-compose.yml` | Docker 配置 | 定义 Directus、Umami、PostgreSQL 服务 |

| `frontend/tailwind.config.js` | Tailwind 配置 | 主题色、字体等样式配置 |

## 🚀 部署方式

### 方式一：宝塔面板（推荐）

```bash
# 1. 创建站点
# 宝塔面板 → 网站 → 添加站点

# 2. 上传前端文件
# 上传 frontend/dist/client 到 /www/wwwroot/b2bbaby

# 3. 配置 Nginx
# 使用 nginx/nginx-baota.conf 配置

# 4. 启动 SSR 服务
cd /www/wwwroot/b2bbaby/frontend
pm2 start dist/server/entry.mjs --name b2bbaby-ssr

# 5. 启动 Docker 服务
cd /www/docker/b2bbaby
docker-compose up -d
```

### 方式二：Docker Compose

```bash
# 启动所有服务
docker-compose up -d

# 查看状态
docker-compose ps

# 查看日志
docker-compose logs -f
```

## 📞 常见问题

### Q1: 前端页面在哪里修改？
A: 修改 `frontend/src/pages/` 下的 `.astro` 文件，然后重新构建。

### Q2: 内容数据在哪里？
A: 博客/产品/FAQ 数据在 `frontend/src/lib/` 下的 `.ts` 文件中，运输证明在 `frontend/src/content/shipping-proof/index.json`。

### Q3: 图片放在哪里？
A: 放在 `frontend/public/images/` 目录下，通过 `/images/xxx.jpg` 访问。

### Q4: 如何更新前端？
A:
```bash
cd /www/wwwroot/b2bbaby/frontend
npm run build
pm2 restart b2bbaby-ssr
```
