## 3. 图片上传

### 3.1 单图片上传

1. 找到带有 **Image** 标签的字段
2. 点击字段区域，会显示 **Upload File** 按钮
3. 点击按钮选择本地图片文件
4. 图片会自动上传并显示预览

### 3.2 多图片上传（Gallery）

1. 找到带有 **Gallery Images** 标签的字段
2. 点击字段区域，会显示上传界面
3. 点击 **Add Files** 按钮选择多个图片
4. 可以拖拽调整图片顺序

### 3.3 图片要求

| 项目 | 要求 |
|------|------|
| 格式 | JPG, PNG, WebP |
| 大小 | 建议 < 2MB |
| 尺寸 | 产品图片建议 800x800 以上 |

## 4. 自动更新

### 4.1 自动更新机制

当您在后台修改任何内容并点击 **保存** 时：
1. 系统会自动触发工作流
2. 工作流调用前端构建 API
3. 前端自动重新构建并部署到公网

**触发条件**:
- 创建新内容（items.create）
- 更新现有内容（items.update）
- 删除内容（items.delete）

**支持的集合**:
- Products（产品）
- FAQs（常见问题）
- Blogs（博客）
- About（关于我们）
- Home（首页）
- OEM/ODM（服务）
- Contact（联系信息）
- Site Settings（网站设置）

### 4.2 手动更新（备用）

如果自动更新未触发，可以手动运行更新脚本：

1. 打开 `newbaby` 文件夹
2. 双击运行 `update-to-production.bat`
3. 等待命令行窗口显示"更新完成"
4. 刷新公网页面查看更改

## 5. 扩展管理

### 5.1 安装扩展

1. 点击左侧菜单 **设置** → **扩展集市**
2. 在搜索框输入想要的扩展名称
3. 找到扩展后点击 **安装** 按钮
4. 安装完成后点击 **启用**

### 5.2 推荐扩展

| 扩展名称 | 类型 | 用途 |
|----------|------|------|
| CKEditor 5 | Interface | 强大的富文本编辑器 |
| Color Picker | Interface | 颜色选择器字段 |
| Repeater Plus | Interface | 增强版可重复字段 |
| Dashboard Widgets | Panel | 数据统计图表 |
| Image Optimizer | Hook | 自动压缩图片 |

### 5.3 管理已安装扩展

1. 点击左侧菜单 **设置** → **扩展**
2. 可以看到所有已安装的扩展
3. 点击扩展可以查看详情或卸载

## 📞 技术支持

如果遇到问题，请检查：

1. **服务状态**: 确保 Docker Compose 正在运行
2. **网络连接**: 确保本地网络正常
3. **浏览器缓存**: 按 `Ctrl + Shift + R` 强制刷新
4. **查看日志**: 查看 Docker 容器日志
